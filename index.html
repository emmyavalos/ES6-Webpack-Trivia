ES6 and Webpack Basics

Facts and triva about ES6 and Webpack.
ECMAScript (ES) is a standard for scripting languages
JavaScript is an implementation of ECMAScript.
ES6 is the newest version and was the most extensive update since 1997.
There are three keywords available to declare a variable - var, let and const.
The ES6 update has arrow functions, a shorter way to declare and use functions
Another benefit of arrow functions is that they provide implicit returns.
Webpack is a module bundler, a tool for bundling application source code
It bundles the modules and generates static assets representing them.
